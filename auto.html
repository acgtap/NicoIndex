    被动功能

    vuePress-theme-reco Yakumo323    2017 - 2023

    被动功能

    Yakumo323

    # 表情回复

    通过添加表情列表的方式我们可以让机器人触发指定词语的时候回复我们的内容。 发送『添加表情xxx』，随后再发送一张图片，当你再次发送xxx时，机器人就会回复你刚刚发送的图片。

    当私聊发送图片，会自动依据图片内容在网上寻找表情包回复。回复概率>90%

    # 戳一戳

    当你戳一戳机器人时，机器人会有概率以某种方式回复。机器人有概率会触发一定的事件。有些事件会导致CD时间，这就可能导致机器人在短时间内可能无法回应你。这是正常现象。

    # 群聊复读机

    当我们正常聊天的时候，机器人会随机的从以前的群聊天记录里面随机出一条消息发送。群聊超过一小时没有聊天的时候，可能会随机从以前的群聊天记录里面随机出一条消息发送。以上功能需要在已经群领养的条件下才能发挥作用，其他没有领养的群，没有这种被动效果。

    # 悄悄话

    当你天天与机器人私聊时，有概率在每天下午收到AI的悄悄话，内容主要以对事物看法之类的。目前已调低概率，每天最多一次悄悄话。

    # 私聊体验

    派蒙更加注重私聊体验，聊天会自动斗图，会根据表情包不一样发送不一样的表情包，当你天天与机器人私聊时，在过节会收到机器人的祝福（仅主AI），偶尔也会收到礼物，同时当你和机器人有了火花后，且开启幸运字符后，机器人就会每天自动抽取火花（仅主AI）。如果你前一天与机器人有过聊天，机器人就会给你发送早晚安，且随机可能会找话题与你聊天，或与你说瞧瞧话。 目前可以通过『建立闺蜜关系』『建立基友关系』『建立死党关系』来建立关系，此关系是基于QQ的亲密关系，建立后你们可以解锁更多交流互动。

    【注意】在私聊中，绝大多数命令无法使用，你只能与机器人聊天，或者斗图。当然也存在一些指令可用，比如十连、绑定uid等，这里不一一列出。常见的如菜单、签到、钱包、pet等私聊都无法使用，且私聊不支持at匹配，即有at会自动过滤，也可能出现不知道怎么答复的回复。

    # 好感度

    主要用于自我保护，检测攻击性话语等用的，计算的好感其实不是很准。

    喵呜会对你说的话进行情感分析，根据正负情况计算好感。 为正会加好感，为负时会检测负面关键词决定是否进行减好感。 好感值过低会对你历史说过的话进行审查，确认有攻击性的话语等会进行拉黑处理。 好感仅仅是用于辅助判断，绝对不会影响其他功能使用权限， 好感高低不会有特别对待，不要刻意刷好感， 好感度低也不用担心，只要没有对喵呜说过攻击性的话，一般是不会拉黑的， 正常使用就好，如果不喜欢机器人可以屏蔽掉，还请不要随便骂。

    # 黑名单

    黑名单会记录一些对机器人不友好的人，对于黑名单成员，喵呜不会回复他们的任何消息。

    # 违禁词检测

    喵呜会检测向喵呜发送的消息中的违禁词，若有违禁词则可能会禁言触发者， 并且会拉黑10分钟以及转告主人，会视情况决定是否永久拉黑，但是误触发不会被永久拉黑

    # 防刷屏

    发消息过快时（4s内发送超过5条消息），喵呜会不理刷屏者 30s ，主要用于防止机器人之间互相刷屏

    # 被禁言后行为

    喵呜被禁言后会主动退群且将群拉入黑名单 再次拉入禁言的群也不回复被禁言的群里的消息，即便已经解禁）

    需要解禁在主群联系群主。

    # 复读机

    随机复读群友之前发的一句话
