    # 家族争霸

    （1）创建家族+家族名：创建一个家族，不消耗金币

    （2）加入家族+家族名：加入一个家族

    （3）我的家族：查看我当前所加入的家族

    （4）查看家族+家族名：查看其他家族

    （5）家族列表：列出所有家族

    （6）家族排行：依照家族基金从多到少排序

    （7）家族捐赠+金币数量：捐金币，提升家族基金

    （8）家族福利：每天可以领取金币，数量是家族基金的十分之一

    （9）退出家族：退出当前所在家族

    （10）解散家族：解散家族，请不要乱输入此命令

    （11）家族公告+一句话：更改家族宣言

    # 圣林农场

    （1）种子商店：查看所有可以种植的种子

    （2）种植+种子名：使用一定金币种植，种植后收获时间随机2-5小时

    （3）收获+种子名：到收获时间就可以使用命令收获种子

    （4）种植信息：查看你所有的种的种子

    （5）抢菜+@：可以抢别人的菜，与魔法大陆联动

    # 大头菜买卖交易市场

    玩法： 解锁大头菜收购服务后，每个星期天就可以买到大头菜啦，随后星期一到星期六大头菜的价格每半天都会变一次，这个时候你可选择卖出或继续等待下一次价格变化， 收购的时候记得记下大头菜的单价。

    注意： 解锁大头菜收购服务需要10000铃钱（金币）； 由于大头菜当天不能立即出售，所以找个地方放好，记住要在7天内出售，不然大头菜会坏掉； 坏掉的大头菜在储物柜中呈现灰色，卖掉只能得到很少的铃钱； 给予可以将储物柜里的大头菜给别人。

    命令： 解锁大头菜收购服务、今天的大头菜价格、

    买大头菜xx、卖大头菜xx，给予@群友 1

    # 魔法大陆

    1. 收集魔法糖：每天可以收集一次魔法糖
    2. 炼铜+魔法糖数量：炼铜成功可以将魔法糖翻倍
    3. 炼金+魔法糖数量：炼金成功可以将魔法糖三倍
    4. 魔法商店：查看所有魔法道具
    5. 兑换+道具名：用魔法糖兑换魔法商店理的道具
    6. 魔法攻击+at一个人：使用魔法攻击，成功率与自己和对方魔法数据有关
    7. 魔法师等级：查看魔法师等级，魔法师等级经验由魔法攻击时获得
    8. 领取等级奖励：魔法师等级上升一级后可以使用此命令领取奖励
    9. 我的魔法糖：快速查询我的剩余魔法糖

    # 援力交易所

    本援力系统的目的，是通过市值的方式，动态的展示不同虚拟人物的人气高低。

    你可能需要快速的游览下面的一些基础设定，来帮助你尽快的融入这个游戏。

    # 速查词典

    **「援力」**是援力系统唯一指定货币。伴随着角色被充能后产生的一种神秘力量。你可以使用这种力量去和其他玩家交易。

    「股票」（股份）是「许愿系统」向粉丝发放一种可转凭证，拥有这种凭证的多少，象征着你对这个角色的贡献的多少。你可以使用「援力」买入，也可以卖出这种凭证。

    **「许愿」**粉丝通过「许愿」向自己喜欢的角色传达爱意。在这个过程中，「许愿」会自动向粉丝发放「股票」奖励并提高角色市值

    **「交易」**由于「股票」具有可以转让的特性，粉丝们可以通过「交易所」提供的交易功能互相交易自己喜欢的角色。

    「喊话」「许愿」和「交易」的时候可以附加一句话作为「喊话」。你可以用来表达你的态度，或者回应别人的「喊话」，请尽量使用。

    # 命令

    1. 我的入股：查看我的入股情况

    2. 角色列表：列出所有的角色

    3. 查看角色+名称：查看角色的具体信息，比如市值，代号，股票

    4. 许愿+角色代号+#+许愿次数：许愿来获得股票，单次许愿需要100援力。注：角色代号可以通过查看角色获得。

      使用举例：许愿KM#100 含义：许愿加藤惠100次

    5. 市值排行：将市值从高到低进行排序

    6. 股票排行：将角色的股票数进行排序

    援力可以通过道具商店购买。命令：购买援力

    游戏流程：目前在初期，难度不高，主要以许愿为主，交易所暂时不开放
