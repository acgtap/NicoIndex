    # 维基萌抽卡玩家教程

    # 介绍

    维基萌抽卡是一款由玩家自由DIY卡牌的卡牌收集游戏。

    # 快速入门教程

    1. 新玩家在注册后会获得【结缘币】，在菜单选择【结缘】后，进入结缘界面。
    2. 在结缘界面选择卡包和相对应的结缘次数后可以获得对应的卡牌。
    3. 如果获得的卡牌超过20种的话就可以在【对战】→【组建对战卡牌】中编辑自己的对战卡牌。
    4. 尝试在【对战】中点击【匹配对手（普通）】来进行对战。
    5. 在空闲时间可以点击【矿场】进行挖矿，挖出来的数字代表周围一圈里有多少星星矿。
    6. 如果挖到星星矿则可以获得一定量的星星，如果只是数字的话，可以获得对应的宝石。
    7. 宝石可以在【对战】→【升级我的对战卡牌】中对卡牌进行【升级卡牌】。
    8. 星星可以在【商店】中进行抽卡，或者在【市场】中购买自己想要的卡牌。

    # 玩家信息


    在登录游戏后，顶部会出现玩家信息栏。
    从左到右依次是【头像】、【昵称】、【持有的星星】、【导航】
    点击【导航】会显示和首页菜单一样的导航内容。

    这个功能主要是方便手机玩家进行页面切换。

    点击玩家头像后会弹出玩家的详细信息

    可以在这里查看当前的统计信息。
    点击【分享】按钮会自动生成并复制玩家的卡牌链接，玩家可以通过分享该链接让所有人看到自己的卡牌收集情况！
    点击【登出】按钮可以退出当前账号。

    点击弹窗的玩家信息弹窗中的头像，可以修改玩家头像。

    在弹出的玩家修改头像弹窗中，玩家可以根据自己的需求调整自己的头像。
    点击【保存】按钮，即可保存头像。
    注意：本功能因为使用了WebGL功能，所以需要检查浏览器或者系统是否支持WebGL!
    另外也可以通过gravatar来显示头像。
    如果玩家注册的邮箱地址在gravatar中也注册了的话，那么无需重新设置头像，系统将默认显示gravatar的头像。
    只要没有上传过头像，那么默认头像将会显示gravatar的头像。

    # 日常抽卡


    玩家每天都可以通过日常抽卡来获取卡牌。

    在卡包的下拉栏中，玩家可以确认每种卡包当前的收集情况（已收集的卡牌种类/该卡包下一共有多少种卡牌） 选择卡包，选择下面3张卡牌的其中1张，即可获得该卡牌。

    抽完卡后，系统会高亮玩家所选择的那张卡牌，如果是未获取过的卡牌，会在左下角出现【NEW】的标签。

    # 置顶公告


    在菜单栏上方会显示置顶公告，这里可以查看管理员发布的重要事项，务必时刻关注下。

    # 菜单


    玩家可以通过点击右上角导航按钮或者页面中央的菜单来去想要的界面。
    其中【签到】会在当日未签到的情况下右上角出现红点，【邮件】会在有未领取的邮件时右上角出现红点。

    # 排行榜


    排行榜界面中可以查看各种统计数据下的玩家排行情况。
    正常情况下一天更新一次。
    点击每列信息都会展示该大佬的卡牌信息！

    # 最新动态


    此区域用于展示一些当前发生的一些事件。
    例如某位大佬【使用星星】、【抽到卡牌】、【卡牌分解】、【挖到星星】、【挑战动态】等等。
    点击每列信息的头像都会展示该大佬的卡牌信息！

    # 商店


    在商店中可以使用通货【星星】来进行抽卡。
    选择上面的卡包和下面的连抽次数即可抽卡。
    其中50连抽和100连抽都有对应的保底6星卡牌。

    如果是新获得的卡牌，会在卡牌左下角显示一个【NEW】的标签。

    # 结缘


    【结缘】和【商店】类似，都是使用通货进行抽卡，只不过【商店】是使用【星星】，而【结缘】使用的是【结缘币】，其中【结缘】并不保底。同样，如果是新获得的卡牌也会在左下角显示一个【NEW】的标签。

    # 矿场


    其实所谓的【星星矿场】就是一个扫雷游戏。不一样的是扫雷是要躲避地雷，而【星星矿场】则是找到星星（地雷）然后获得星星。也就是说和扫雷是相反的游戏方式。
    当然，没有挖到星星也没有关系，系统会根据挖出来的数字给与一定的【宝石】奖励。而【宝石】是作为【升级卡牌】的重要道具！

    对于很多没有接触过扫雷的大佬，可以去网上搜一搜扫雷的相关教程。
    例如<手把手教你玩扫雷> (opens new window)。

    上方的【当前在线】可以知道矿场有多少玩家在线，点击后面图标可以查看在线玩家信息。


    鼠标移动，或者点击【星星矿场】标题，可以查看当前矿场高产出的宝石。


    每一种镐有其不同的属性：
    铁镐：一把简单的镐，十分钟就能做出来，但是能成功挖出来的星星矿并不多！
    银镐：银质的镐，制作需要半小时，挖出来的星星矿至少比铁镐多！
    金镐：制作工艺非常难的镐，制作需要一小时，金镐能最大程度减少挖掘对星星矿造成的破坏，所以挖出来的星星矿产量非常高！

    原则上应该使用铁镐和银镐去探矿，然后用金镐去铲星星，这样就可以使星星收获最大化。


    除了以上三种镐还有另外两种特殊镐和保罗炸弹。
    星星镐：仅能挖出星星的镐，挖出来的星星略微比金镐多，但是如果挖到宝石则什么都不会获得。
    宝石镐：仅能挖出宝石的镐，挖出来的宝石略微比金镐多，但是如果挖到星星则什么都不会获得。
    保罗炸弹：能一次性轰开3×3片矿区，但是因为其爆炸力惊人，只能保留少量的宝石，星星则完全被炸毁。

    每张矿场地图的最后一片星星矿产出将会是原来的三倍！如果有十分把握的话，推荐用星星镐来挖最后的星星矿！

    有时候挖矿会遇到需要计算验证码的情况，此时只要验证码的错误次数不要太多，就会多的一定量的【神秘碎片】。

    【神秘碎片】可以在【神秘碎片商店】中兑换对应的道具。

    # 界面说明


    玩家可以在对战界面中【对战】、【组建卡牌】、【升级卡牌】、【查看往期对战统计信息】、【查看或回放对战】。

    对战界面大致分为4个区域。
    【玩家当前对战信息】、【功能按钮】、【生涯统计】、【对战记录】。

    在玩家对战信息中玩家可以查看自己在当前月份的【胜负信息】、【竞技点】、【对战机会】。

    功能按钮中可以选择对应的功能。
    跳过战斗动画：接下来的对战将直接显示结果。 匹配对手（普通）：这里系统会为玩家匹配一位竞技点与玩家相近的玩家进行对战。
    匹配对手（进阶）：如果玩家觉得现在匹配的对手太弱了，可以选择进阶匹配。进阶匹配只会匹配比玩家竞技点高很多的对手。
    组建我的对战卡牌：进入【组建对战卡牌】界面。
    升级我的对战卡牌：进入【升级卡牌】界面。

    生涯统计中可以查看玩家往期的对战统计信息。
    在目前的服务器中，以一个月为单位，重置玩家的【竞技点】、【胜负信息】，并结算【结缘币】。

    在对战记录中可以查看自己和对手的对战信息。

    点击头像会出现会出现玩家的对战属性和玩家使用的卡牌及其等级。
    其中显示的对战卡牌信息仅限对战结束前所使用的卡牌，并不会显示该玩家的所有对战卡牌信息。

    # 竞技点

    服务器每个月的1号会对所有玩家的对战信息进行重置。
    玩家的【竞技点】会以500为一个单位进行重置，并结算【结缘币】。
    只要玩家的【竞技点】高于500分就会获得对应的【结缘币】奖励。
    如果玩家的【竞技点】排名靠前，可以获得额外的【结缘币】奖励。
    注意：如果玩家在本月没有任何对战记录的话，系统将不会为玩家结算【结缘币】。

    # 组建对战卡牌

    当玩家持有的卡牌超过20种的时候，可以尝试组建自己的对战卡牌。

    在界面的头部，可以查看当前配置的卡牌的【攻】、【防】、【速】、【SAN】的信息。
    攻：影响玩家卡牌的攻击力。
    防：影响玩家卡牌的防御力。
    速：速度大的玩家优先攻击，如果一样则随机。 SAN：玩家生命值，归零则战败。

    配卡区域的数字代表攻击顺序，玩家可以按住移动图标对卡牌进行排序。
    点击数字或者已经配置的卡牌，可以给这个顺序配置一张卡牌。

    配置完对战卡牌后记得点击保存，将卡牌信息保存至服务器。
    另外通过更多按钮可以查看配卡的【说明】、【清空】所有对战卡牌、进入【升级卡牌】界面。

    配卡时，请务必仔细阅读【配卡说明】！

    # 升级卡牌


    升级卡牌会对应所升级卡牌的对应属性，给与不同属性的加成，具体加成情况可以点击【查看属性加成】中确认。

    升级对战卡牌界面默认显示对战卡牌，如果需要升级对战卡牌以外的卡牌，请将【仅显示对战卡牌】的勾选去掉。

    升级卡牌需要一样的【卡牌】和【宝石】。
    【宝石】可以通过【矿场】获得。
    【卡牌】就需要通过【抽卡】和【市场】中获得了。
    如果没有一样的【卡牌】也没有关系，可以通过对应星星的【卡牌碎片】来替代【卡牌】。
    【卡牌碎片】可以在【卡牌分解】中获得。


    另外点击卡牌，可以查看卡牌详情，并且可以快速进行市场的【购卡】和【求卡】。

    点击【升级】按钮右侧的图标可以进入【卡牌等级转换】对该卡牌进行等级转换。

    # 卡牌等级转换


    玩家可以通过【等级转换卷轴】对卡牌的等级进行互换。
    注意：互换只能在同星级和同被动属性（即左上角图标一样）的卡牌之间进行互换。

    # 猜卡

    系统会在每个整点生成一组猜卡卡牌，玩家可以在里面选择6张卡牌。
    在下一个整点系统会生成新的猜卡卡牌，并抽选上个整点的6张卡牌。
    如果玩家选中的卡牌和系统自动抽选的卡牌一样，就会获得该卡牌和对应的奖励。
    猜卡每天第一次是免费的，往后会需要支付对应的星星才能猜卡。
    在猜卡中，玩家可以获得猜卡限定的卡牌。
    在猜卡的卡牌中会显示【出自卡包】和【持有数量】，如果该卡牌为玩家的对战卡牌则还会显示一个【战】字代表是玩家的对战卡牌。

    # 市场

    • 买卡
    • 卖卡
    • 求卡
    # 买卡

    玩家可以在买卡中使用【星星】买取自己需要的卡牌。

    卡牌右下角的勾意味着该卡牌是否拥有。
    玩家也可以通过上面的筛选，直接筛选自己未拥有的卡牌。

    卡包下拉栏中将会显示该卡包下一共有多少张卡牌在贩卖

    # 卖卡

    玩家可以将多余的卡牌挂在市场进行贩卖。贩卖成功后，系统将会收取一定费用。
    贩卖的卡牌有效期为3个月，超过3个月后卡牌将会过期，此时需要玩家手动【更新】卖卡信息才能重新显示在【买卡】中。


    在【可卖】的卡牌中，可以确认自己能卖哪些卡。
    由于系统会对每种卡保留1张，所以仅显示多余的卡牌。
    卡牌的右下角会显示【x人想要】，意味着这张卡牌有多少人在【求卡】中发布了求卡信息。
    玩家也可以直接通过上面的筛选，直接筛选出【是否求购】的卡牌。

    # 求卡

    玩家可以在【求卡】中确认全部求卡信息。
    在上方的筛选中将【我的】改为是，则显示自己的求卡信息。
    如果玩家拥有该卡牌，就会出现【去卖卡】的按钮，快速进入卖卡界面。

    卡包下拉栏会显示该卡包下的求卡数（包括自己的）

    下方的【发布求购】按钮，会进入【图鉴】界面，默认显示的是未拥有的卡牌，如果需要显示全部卡牌，请在筛选栏的【是否拥有】中选择【全部】。 在【图鉴】界面中，点击卡牌即可对该卡牌进行求购。

    # 分解

    • 卡牌分解
    • 道具分解
    # 卡牌分解


    在卡牌分解中，系统会自动保留一张卡牌，并且默认显示不参战的卡牌。
    在卡牌中会显示当前可以分解的数量和【锁定】按钮，点击【未锁定】可以锁定卡牌防止被分解。
    下方输入的数字是要分解的数量。
    当然也可以点击下方【全页】按钮对本页所有的卡牌进行分解。

    分解将会获得对应的【星星】和【卡牌碎片】。
    【卡牌碎片】可以在【升级卡牌】中替代消耗的卡牌。

    # 道具分解

    道具分解中可以将不用的道具分解成【星星】。
    每种道具需要满1000才能进行分解。

    # 图鉴


    在图鉴中，玩家可以知道每种卡包的收集情况以及哪些卡未收集到。

    点击卡牌可以查看卡牌的详情，以及快速在市场购买该卡牌或者求卡。


    点击上方的【批量求卡】可以进入批量求卡状态。
    勾选需要求的卡牌，即可批量对卡牌（最多60种）进行求卡。

    # 制卡

    在制卡中，玩家可以自己DIY卡牌并上传。
    在审核成功后，卡牌进入卡包，所有玩家都将有机会获得。当然投稿的玩家也将获得【星星】奖励！
    注意：请仔细阅读制卡说明后投稿！

    # 签到

    玩家每天可以通过签到获得不同的道具或者星星奖励！
    如果玩家当天忘记签到，可以通过道具【补签羽毛笔】进行补签。

    # 任务

    玩家可以通过任务提示获得对应的奖励。
    每种任务都有过期时间，但是接受任务之后任务会被锁定，想要放弃的话需要支付一定星星才能解除，所以接受任务时一定要保证能够完成。
    每完成30次任务后，可以在下面领取额外的奖励！

    # 邮件

    玩家可以通过邮件来领取系统所赠送的奖励。
    邮件都有有效期，超过有效期将无法领取！所以如果看到邮件图标上有小红点的话，请及时领取！

    # 公告

    玩家可以在公告中确认管理员所发布过的所有公告信息。
