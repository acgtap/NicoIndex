    # 早安/晚安

    @妮可 及妮可安/晚安 会根据时间问好(问坏x)

    类似命令还有：早上好、中午好、下午好、晚上好

    示例：
    A   : @妮可 早
    妮可：@A 早上好呀！

    2
    3

    # pet@某人

    生成各种奇怪的GIF
    使用方法: 发送 pet@某人即可

    该功能详细用法请见功能指引 Petpet

    # 抽签

    查看你今天的运势
    使用方法: 发送 抽签 即可

    # 摸、丢、牵、赞@某人

    （1）丢@某人：这个命令可以生成一张丢出对方头像的图片

    （2）摸/摸爆+@或摸我：这个命令可以生成一张摸出对方或自己头像的图片

    （3）锤/锤爆+@某人：这个命令可以生成一张锤对方头像的gif图片

    （4）牵@某人：这个命令可以生成一张牵对方头像的图片

    （5）赞@某人：这个命令可以生成一张赞对方头像的图片

    该功能详细用法请见功能指引 Petpet

    # 哪个群友是我老婆

    随机抽取一位群友成为你的老婆

    • 用法：
      发送哪个群友是我老婆或谁是我老婆会随机抽取一位群友成为你的老婆~
    • 示例：
      群友A：哪个群友是我老婆
      妮可 ：@群友A 今天你的群老婆是 ...

    # 跟我说

    在命令后面接上一个句子，会语音合成并发送一条语音。

    # 亚托莉说

    在命令后面接上一个日语句子，会语音合成并发送一条语音。

    # 黑历史

    一个用于记录群成员说过的话的功能

    你是否为群成员满口骚话却无法有效记录而困扰？ 那这便是为你而准备的一个功能。

    玩法

    调用指令

    黑历史 名字 @xxx
    1

    来注册一个对他人说话的监听

    之后再群内发送 名字 时便就会返回他人自监听开始的某一句话

    # 来份涩图

    只在部分群开启且对不同AI有不同要求，要开启需要申请，可以在主群进行询问

    功能：发送随机二次元图

    触发方式：发送来点好康的xxx，随机色图 即可

    # 来个猫猫

    功能：随机发送一张猫猫图gif

    触发方式：发送来个猫猫，随机猫猫 即可

    # 舔狗日记

    功能：发送一则舔狗日记

    触发方式：发送舔狗日记，舔

    # 土味情话

    功能：发送一则土味情话

    触发方式：发送土味情话即可

    # 随机文案

    功能：发送网抑云文案、一些有趣的句子、一些伤感的句子、一些关于友情、爱情、生活、未来的句子。希望能从这些句子中得到治愈。

    触发方式：发送随机文案，句子即可

    # 随机图片

    功能：发送随机二次元图

    触发方式：发送 随机图片 即可

    # 天弃之子

    随机群员禁言

    触发方式：发送 天弃之子 即可

    # 生成假消息

    allinall+消息：生成100名随机群员（群人数>=100）的假消息，若群人数<100则生成所有群员的假消息

    oneinall+@+消息：生成指定群员假消息

    randominall+消息：随机群员假消息
