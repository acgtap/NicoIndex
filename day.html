    # 附录

    # 回忆值

    机器人和英雄之间的回忆。将会根据活跃天数和发言数得出，好感度上升也会增加若干回忆值，回忆值也会决定好感上限。该数值每位用户独立拥有。

    # 相识天数

    与机器人相遇到现在的天数。从第一次增加好感起或签到每满24小时计算1天，会对关系和回忆产生影响。该数值每位用户独立拥有。

    # 金币

    机器人的专用货币。可以通过打卡和小游戏、特典等方式获取，用于购买商城道具或和游戏。该数值每位用户独立拥有，通过与机器人相处获得，不存在“付费氪金”这样的方式。

    # 年龄

    AI的年龄，机器人从出生起至今的天数。该数值所有用户共同拥有。

    # 智慧

    AI的智慧值，机器人每学会一个新词条增加100点智慧。该数值所有用户共同拥有。将来会增加到状态中。
