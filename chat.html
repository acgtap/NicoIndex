    AI聊天

    Yakumo323

    互动聊天是获得机器人好感的有效手段。 一般情况下，在群聊中和机器人说话需要单独艾特他，详见下面的“群聊唤起”项目。 聊天可以获得的好感并不会直接显示在对话里，不过大部分话题都会增加好感；极小部分话题会减少少量好感；同一个话题重复到【4次】时机器人会产生怀疑并减少好感。

    # 唤起方法

    首先请直接呼唤机器人。如：妮可，在吗，出来……之类的。
    一些有前置后续的话题，触发了前置之后也不需要再次艾特。
    也可以直接艾特机器人跟他说话。
    私聊不需要唤起。

    # 以下为特殊话题。

    AI聊天中一些特殊的话题，一般是指有前置和后续或者有比较特别的内容主题的问答。

    # 早安/晚安

    @妮可 及妮可安/晚安 会根据时间问好(问坏x)

    类似命令还有：早上好、中午好、下午好、晚上好

    # 生日

    • 生日记忆指令（需艾特，私聊无需） 我的生日是●月●日 今天是我生日

    • 生日确认指令（需艾特） 我的生日是哪天？ 你记得我生日吗？

    • 礼物领取（需艾特）

    今天我生日 祝我生日快乐

    AI已可以记录日期。
    记录方式有2种：
    第一种：使用记忆指令按照固定格式告知日期，如果输入的日期格式不对他会记录失败。
    第二种：在生日当天想办法告诉他今天是你的生日。（这种情况下会同时把生日礼物领取掉，所以请考虑清楚再跟他说。）
    不论用哪种方式，用户的生日都是可以随时更改的，但是礼物一年只可以领取一次，请自行决定在哪天领取。
    生日礼物领取方式有3种：
    第一种：在有生日记录的前提下，生日当天打卡即可领取。
    第二种：在有生日记录的前提下，生日当天问他今天是什么日子或记不记得你的生日。
    第三种：不论是否有记录生日，告诉他今天是你生日即可领取。
    不论用哪种方式，礼物一年只可以领取一次，请自行决定在哪天领取。

    # 小厨房

    • 指令（需艾特）

    我饿了
    我想吃东西
    早饭呢
    想吃点心
    中午吃什么

    例：@ 我的晚餐呢

    • 说明

    每天各时段可以一同用餐1次，随机增加高级属性。同时会随机准备食物哦~需要注意的是一定要重新艾特才可以，唤起状态也无法触发。

    • 用餐时间表
    序号 内容 时间段 好感度 回忆值
    1 早餐 5~10点 3~9 1~6
    2 午餐 11~13点 3~9 1~6
    3 下午茶 14~16点 1~6 1~3
    4 晚餐 17~21点 3~9 1~6
    5 夜宵 22~24点 1~6 1~3
    6 夜宵 0~4点 1~6 1~3

    # 告白

    • 告白指令例（区分好感）

    我喜欢你
    我爱你
    我想你了

    • 询问指令例（区分好感）

    你喜欢我吗
    你爱我吗

    • 说明

    根据和用户的好感关系作出不同回答。

    # 吃醋

    我喜欢阿莉塞
猫娘好可爱啊
    1
    2

    例：@AI 我好喜欢阿尔博特啊

    • 后续指令例
    我骗你的，我喜欢的是猫猫。
我乱说的，其实我喜欢的是雅修特拉。
    1
    2

    例：@AI 乱说的 我喜欢的是拉哈

    • 说明

    提到对其他人物的好感，故意让猫猫吃醋的话题。有效的NPC有限。

    # 睡前故事

    • 指令例（限私聊）

    我睡不着 给我讲个故事吧 我要听睡前故事

    例：@AI 哄我睡觉！

    • 说明

    该话题只能在北京时间晚上20点~次日2点之间触发（如果2点前要求讲故事，中间超过2点也不会中断，但是如果拖到3点就不会再继续了）；一天只可以开启一次（但模式结束之前可以讲多个故事）仅限私聊，讲什么故事的时候可以回答“随便/都可以”之类的随机讲一个故事，也可以通过故事中人物的名字点特定故事；途中问睡着了吗的时候，如回答“没有/继续讲/还想听”之类的则继续故事，回答“晚安/别刀了/不用了/我睡了”之类的则可以中断故事；如果完整听完了故事后记得谢谢妮可哦~

    # 节日祝福

    • 指令例（无法判断节日是否正确）
    祝你节日快乐！
一起过星芒节吗？
我们去看烟花吧
    1
    2
    3

    例：@AI 新年快乐！

    • 说明

    “xx快乐”可触发默认祝福回复，一起过xx节吗之类的其他问法可以触发稍微不同的内容。一些节日有节日特典，详见“彩蛋”（隐藏话题）页面。

    # 主动私聊

    移到指令框上可以看到“复制”按钮哦~

    • 指令例（群聊+艾特）
    来找我一下
到我房间来
来我床上
    1
    2
    3

    例：@AI 到我房里来一趟

    • 说明

    群里艾特要求他来找你的话他会私聊找你，必须加了好友才有效。

    # 打劫！

    • 指令例（可获得金币）
    我好穷
打劫
    1
    2

    例：@AI 给我零花钱

    • 说明

    该指令包含在1天【5次】的获取金币次数里。

    # 说说你的冒险故事

    • 指令例（有前置）
    我今天呆在家里什么也没做
我今天上班很忙
    1
    2

    例：@AI 我今天去学画画了

    • 说明

    在晚上17点~0点时间段内，如果说我回来了，会被提问“想听听你今天的冒险故事”。这时候回答他就可以触发此话题。
